# Netflix Movie Recommender

A Netflix-style movie recommendation system built using:

- Python
- Flask
- Scikit-Learn
- Pandas
- MovieLens Dataset

## Features

- Content-based movie recommendation
- Fuzzy movie search
- Rating-based ranking
- Modern Netflix-inspired UI
- Flask web application

## Dataset

MovieLens Dataset

## Installation

```bash
git clone https://github.com/charusri286/Netflix-recommender.git

cd Netflix-recommender

pip install -r requirements.txt

python app.py
